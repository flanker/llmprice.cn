# 大模型价格对比

## 简介

这个文档的目的是对比国内各个厂商的大模型，主要针对文本生成类模型。

对比内容会包括：

* 模型能力
* 价格（输入价格、输出价格）
* 文本/Token 转化率
* 支持上下文的长度（输出是否有限制）
* Rate Limit 限流情况
* 以及其他一些备注

国内大模型厂商

* 大厂：阿里通义千问，百度文心一言，字节豆包，腾讯混元
* 初创：月之暗面Moonshot，百川，智谱，MiniMax，零一万物
